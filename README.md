# Africa

<h1> About </h1> 
This beautiful SwiftUI App shows videos, images, and locations of animals in Africa.  

<h1> Featuring </h1> 
* Parsing JSON files <br>
* MapKit Integration <br>
* Complete launch screen and app icons <br>
* Grid Layouts <br> 
* Mac Catalyst technology <br>
